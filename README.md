 Helpdesk Automation System

This project automates helpdesk request handling using Google Forms, Google Sheets, and Slack integration.

 Features
- Captures helpdesk requests via form
- Sends confirmation email automatically
- Sends real-time Slack notifications using APIs
- Stores tickets in Google Sheets

 How it Works
1. User submits helpdesk form
2. Data is stored in Google Sheets
3. Apps Script triggers automatically
4. System:
   - Sends confirmation email
   - Sends Slack alert to team
